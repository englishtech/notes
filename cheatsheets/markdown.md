# Шпаргалка по Markdown

## Заголовки

```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

---

## Текст

```markdown
**Жирный**
*Курсив*
~~Зачёркнутый~~
`Код в строке`
```

---

## Списки

### Маркированный
```markdown
- Элемент 1
- Элемент 2
  - Вложенный
```

### Нумерованный
```markdown
1. Первый
2. Второй
3. Третий
```

---

## Ссылки и изображения

```markdown
[Текст ссылки](https://example.com)
![Альтернативный текст](путь/к/изображению.jpg)
```
