# Строки и срезы

## Методы строк

| Метод | Описание | Пример |
|-------|----------|--------|
| `.upper()` | В верхний регистр | `"привет".upper()` → `"ПРИВЕТ"` |
| `.lower()` | В нижний регистр | `"ПРИВЕТ".lower()` → `"привет"` |
| `.title()` | Каждое слово с заглавной | `"hello world".title()` → `"Hello World"` |
| `.capitalize()` | Первую букву заглавной | `"привет".capitalize()` → `"Привет"` |
| `.strip()` | Удалить пробелы по краям | `"  х  ".strip()` → `"х"` |
| `.lstrip()` | Удалить слева | `"  х".lstrip()` → `"х"` |
| `.rstrip()` | Удалить справа | `"х  ".rstrip()` → `"х"` |
| `.replace(old, new)` | Заменить подстроку | `"abc".replace("a", "x")` → `"xbc"` |
| `.split(sep)` | Разбить в список | `"a,b,c".split(",")` → `["a", "b", "c"]` |
| `.rsplit(sep, maxsplit=1)` | Разбить справа | `"a.b.c".rsplit(".", 1)` → `["a.b", "c"]` |
| `.splitlines()` | Разбить по строкам | `"а\nб".splitlines()` → `["а", "б"]` |
| `.join(iterable)` | Объединить строки | `",".join(["a", "b"])` → `"a,b"` |
| `.find(sub)` | Индекс подстроки (-1 если нет) | `"абв".find("б")` → `1` |
| `.rfind(sub)` | Индекс справа | `"ааа".rfind("а")` → `2` |
| `.index(sub)` | Индекс (ошибка если нет) | `"абв".index("б")` → `1` |
| `.startswith(prefix)` | Начинается с | `"http".startswith("h")` → `True` |
| `.endswith(suffix)` | Заканчивается на | `"file.txt".endswith(".txt")` → `True` |
| `.count(sub)` | Сколько раз встречается | `"ааа".count("а")` → `3` |
| `.isdigit()` | Только цифры | `"123".isdigit()` → `True` |
| `.isalpha()` | Только буквы | `"абв".isalpha()` → `True` |
| `.isalnum()` | Буквы и цифры | `"а1б2".isalnum()` → `True` |
| `.isspace()` | Только пробелы | `"  ".isspace()` → `True` |
| `.isupper()` | В верхнем регистре | `"ПРИВЕТ".isupper()` → `True` |
| `.islower()` | В нижнем регистре | `"привет".islower()` → `True` |
| `.center(width)` | Центрировать | `"х".center(5)` → `"  х  "` |
| `.ljust(width)` | Выровнять влево | `"х".ljust(5)` → `"х    "` |
| `.rjust(width)` | Выровнять вправо | `"х".rjust(5)` → `"    х"` |
| `.zfill(width)` | Дополнить нулями | `"5".zfill(3)` → `"005"` |

## Срезы строк

| Синтаксис | Описание | Пример |
|-----------|----------|--------|
| `s[start:stop]` | От `start` до `stop` (не включая) | `"привет"[1:4]` → `"рив"` |
| `s[:stop]` | От начала до `stop` | `"привет"[:3]` → `"при"` |
| `s[start:]` | От `start` до конца | `"привет"[3:]` → `"вет"` |
| `s[:]` | Копия строки | `"х"[:]` → `"х"` |
| `s[::step]` | С шагом | `"привет"[::2]` → `"пие"` |
| `s[::-1]` | Реверс строки | `"привет"[::-1]` → `"тевирп"` |
| `s[-1]` | Последний символ | `"привет"[-1]` → `"т"` |
| `s[-3:]` | Последние 3 символа | `"привет"[-3:]` → `"вет"` |

### Правила срезов:
- Индексы: `0` — первый, `-1` — последний
- Срез **не вызывает ошибку**, даже если индекс за границей
- `start` включается, `stop` **не включается**
- Шаг может быть отрицательным (реверс)

```python
s = "привет"
print(s[0])      # "п"
print(s[-1])     # "т"
print(s[1:4])    # "рив"
print(s[::-1])   # "тевирп"
