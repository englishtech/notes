## Создание виртуального окружения для проекта на компьютере
- Создать папку проекта.  
- Зайти в папку проекта (`cd папка_проекта` в bash или `Open Folder` в VSCode)
- Создать виртуальное окружение:
```bash
python -m venv .venv
```
- Активировать виртуальное окружение:
```bash
source .venv/Scripts/activate
```
(Просто запуск файла `.venv/Scripts/activate.bat`)  
В терминале (`git bash`) может не отображаться префикс `.venv`  
Для проверки ввести:
```bash
which python
```
Должен отобразиться путь до интерпретатора в виртуальном окружении.  
- Деактивировать виртуальное окружение (например, при смене проекта):
```bash
deactivate
```
Активное виртуальное окружение отключится. Возврат к системному интерпретатору Python.
